<p>In this article we will build a mini blogging platform for software engineers. We will use:</p>
<ul>
<li>Vue.js 3 + composition API</li>
<li>Typescript</li>
<li><code>vuex-smart-module</code></li>
</ul>
<p>It will support authoring articles using markdown, include syntax highlighting via <code>highlight.js</code>.</p>
<p>A simple sample of the composition API is as follows:</p>
